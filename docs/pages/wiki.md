# Wiki (`/wiki`)

**Componente:** `WikiPage` (`src/pages/WikiPage/WikiPage.tsx`)

Página de documentação interna que explica cada **fonte de dados** utilizada na plataforma.

## Seções

- **PNCP:** Portal Nacional de Contratações Públicas
- **TCU:** Tribunal de Contas da União (contas irregulares, inabilitados, inidôneos)
- **TSE:** Arquitetura de dados e queries eleitorais
- **Portal da Transparência:** Dados do governo federal
- **Câmara dos Deputados:** API e dados legislativos
- **Senado Federal:** API e dados legislativos
- **OpenCNPJ:** Dados cadastrais de empresas
- **IBGE:** Dados geográficos e populacionais
- **SICONFI:** Dados contábeis de estados e municípios
- **Repositórios:** Links para os projetos no GitHub

## Dados

- Conteúdo estático em markdown/HTML
- Não consome APIs
- Serve como referência para entendimento das fontes de dados

## Particularidades

- Sidebar de navegação entre seções
- Links externos para os repositórios do projeto
